# pedaudit
# Автоматическое определение поддерживающего тона в педагогическом фидбэке (русский язык)

## Обзор проекта

Проект посвящён автоматическому выявлению поддерживающего (ободряющего) тона в коротких русскоязычных комментариях преподавателей.

Работа начиналась как небольшой эксперимент по педагогическому аудиту и превратилась в пилотное NLP-исследование с ручной разметкой, обучением baseline-моделей, кросс-валидацией и лингвистической интерпретацией результатов.

Основная цель — проверить, можно ли надёжно предсказывать поддерживающий тон на основе поверхностных лексических признаков в небольшом корпусе.

---

## Данные

- ~189 коротких русскоязычных реплик
- Ручная разметка по трём критериям:
  - crit1 — конкретность / практическая направленность  
  - crit2 — инструктивность  
  - crit3 — поддерживающий тон  

Из-за сильного дисбаланса классов и малого числа положительных примеров критерии crit1 и crit2 были исключены из моделирования.  
В финальной версии используется только crit3 (поддерживающий тон).

Датасет хранится в файле: teacher_feedback_examples_ru.csv
Колонки:

- comment — текст комментария  
- crit3 — бинарная метка (1 = поддерживающий, 0 = неподдерживающий)

---

## Метод

### Представление текста

- TF-IDF векторизация  
- униграммы + биграммы  
- max_features = 5000

### Проверенные модели

- Logistic Regression (class_weight="balanced")
- Linear SVM (LinearSVC)
- Complement Naive Bayes
- Random Forest

### Оценка качества

- 5-кратная стратифицированная кросс-валидация
- Метрика: F1-score

---

## Результаты

Средний F1 (± стандартное отклонение):

| Модель                | F1 mean | F1 std |
|----------------------|---------|--------|
| Logistic Regression | ~0.84 | ~0.02 |
| LinearSVC           | ~0.83 | ~0.02 |
| Random Forest       | ~0.77 | ~0.01 |
| ComplementNB        | ~0.75 | ~0.05 |

Линейные модели превосходят деревья решений и Naive Bayes на TF-IDF признаках.

Низкое стандартное отклонение указывает на устойчивость результатов несмотря на небольшой размер корпуса.

## Лингвистический анализ

Коэффициенты модели были проанализированы для выявления языковых маркеров.

Поддерживающий фидбэк характеризуется:

- эмоционально-оценочной лексикой (например, «молодец», «великолепно»)
- интенсификаторами степени («очень», «значительно»)
- обращением ко второму лицу («ты», «тобой», «горжусь»)

Неподдерживающие реплики чаще содержат:

- институциональную лексику
- условные конструкции
- сравнения и отрицательное оформление

Это указывает на различие между аффективно-оценочным и административно-инструктивным регистрами педагогического дискурса.

## Выводы

Эксперимент показывает, что поддерживающий тон в педагогическом фидбэке надёжно определяется по поверхностным лексическим признакам даже на относительно небольшом русскоязычном корпусе.

Классические линейные модели дают сильный и интерпретируемый baseline.

Критерии, связанные с инструктивностью и конкретностью, не поддались моделированию из-за крайнего дисбаланса классов, что указывает на ограничения данных, а не моделей.
